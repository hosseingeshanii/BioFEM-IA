Detailed Description

Per-rank local copy of Loop subdivision patch stencil data.

Built once from the global IBMNodes arrays after DMPlex distribution. All arrays are indexed by local cell index lc in [0, nLocalCells). The DM reference is retained so that downstream functions can query it.
